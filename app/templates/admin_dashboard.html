관리자 통합 입력 화면

`/tester` 입력 화면처럼 각 카드에서 바로 표 형태로 입력하고 저장한다. 페이지 이동 없이 `/admin` 한 화면에서 처리한다. 입력 중에는 브라우저에 자동저장되고, 필수 항목이 완성되면 자동제출된다.
ID 컬럼은 관련 value 입력 시 작성규칙에 맞는 초안이 자동 가이드된다. 필요하면 직접 수정 가능하다.
표의 컬럼명(헤더)을 클릭하면 해당 입력칸으로 스크롤·포커스 이동한다. 선행 입력이 비어 있으면 화면 중앙 비모달 안내 후 먼저 채울 칸으로 이동한다.
